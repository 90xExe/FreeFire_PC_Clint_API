# 🚀 How To Start

## Step 1: Open Software

Run the `.exe` file.

After opening, you will see:

- Match ID Input Box
- Download ZIP Button
- Google Sheet Link Box
- Upload Buttons
- Excel Tools
- Live Result
- Logo Resize Tool

---

# 🎯 Feature 1: Download Match ZIP File

## What It Does

This downloads official match result files directly from Free Fire match database.

## How To Use

1. Enter Match ID  
Example:

123456789012345678

or multiple IDs:

123456789012345678,987654321098765432

2. Click **Download ZIP**

3. Choose where to save file.

## Output

You will get a ZIP file containing CSV files like:

- team_data.csv
- player_data.csv
- stats files

# 📊 Feature 2: Upload ZIP to Google Sheets

## What It Does

Automatically uploads CSV files from ZIP into your Google Sheet.

Each CSV creates its own sheet tab.

## How To Use

1. Download ZIP or Upload ZIP manually  
2. Paste Google Sheet link  
3. Click **Upload to Google Sheets**

## Result Example

Google Sheet Tabs:

- team_data
- player_data
- team_data_total

Perfect for live sharing with staff.

:contentReference[oaicite:3]{index=3}

# 📘 Feature 3: Upload ZIP to Excel

## What It Does

Uploads all CSV files into one Excel workbook.

If sheet already exists:

✅ old sheet clears  
✅ new data inserted

## How To Use

1. Click **Browse Excel**
2. Select or create `.xlsx`
3. Click **Upload to Excel**

## Result

One Excel file with multiple sheets.

Perfect for:

- vMix overlays
- production graphics
- analysis
- reports

# 📡 Feature 4: Live Result Viewer

## What It Does

Shows live result table instantly from Match ID.

## How To Use

1. Click **Live Result**
2. Enter Match ID
3. Click Fetch Result

## You Will See

| Team Name | Kill | Score | Booyah |
|----------|------|-------|--------|

Winning team gets highlighted.

Great for casters & observers.

Check Match ID format.

## If Google Sheet not working:

Make sure service Gmail has access.
